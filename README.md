## 疑问

### 为什么点击右上角“×”无法退出程序？
右上角的“×”是关闭窗口，程序会最小化到右下角，并不会退出程序。如果想退出程序，在右下角icon上右击并选择退出。

![3](https://user-images.githubusercontent.com/6293752/92937441-eb0fec00-f47d-11ea-8265-07eb9bda55bd.gif)

### 离开了写作（工作区），如何返回？
点击右上角的我来logo，可以返回工作区。

### 如何拖动窗口？
1. 拖动右上角的“+”。
2. 拖动左边“模板”和“垃圾桶”下面的大片空白区域。
